# Mistral AI .NET Proof of Concept Applications

A collection of Proof of Concept (PoC) applications demonstrating how to use the **Mistral AI APIs** from .NET applications.

This repository contains sample code developed while exploring the Mistral AI platform, including Chat Completion, Fill-in-the-Middle (FIM), Files, and Models APIs. The projects accompany a series of blog posts documenting lessons learned, implementation approaches, and practical experiments.

## Objectives

The goals of this repository are to:

* Explore the capabilities of the Mistral AI platform.
* Demonstrate integration patterns for .NET applications.
* Provide working examples for common API scenarios.
* Evaluate different Mistral models and endpoints.
* Support the accompanying technical blog series.

## Covered APIs

### Chat Completion

Examples of:

* Synchronous chat completions
* Multi-turn conversations
* System and user prompts
* Model configuration
* Response handling

Documentation:

* <https://docs.mistral.ai/api/endpoint/chat>

### Files

Examples of:

Examples of:

### Models

Examples of:
